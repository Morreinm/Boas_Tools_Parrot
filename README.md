Colecao de ferramentas para pentest

## Ferramentas de Pentest

- **nmap** - Scanner de portas e servicos
- **gobuster** - Bruteforce de diretorios e arquivos
- **sqlmap** - Scanner e exploracao de SQL Injection
- **metasploit** - Framework de exploits
- **hydra** - Bruteforce de senhas
- **nikto** - Scanner de vulnerabilidades web
- **dirb** - Descoberta de diretorios web
- **john** - Quebra de hashes (John the Ripper)
- **hashcat** - Quebra de hashes avancada
- **aircrack-ng** - Auditoria de redes wireless
- **burpsuite** - Proxy e scanner web
- **netcat** - Ferramenta de conexao universal
